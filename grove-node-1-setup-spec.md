# Grove Dev Machine Setup Spec
## Target: grove-node-1

**Purpose:** Configure this machine as a fully functional Grove development environment with Claude Code.

---

## Phase 1: Core Development Tools

### 1.1 Git
- **Install:** Git for Windows from https://git-scm.com/download/win
- **Verify:** `git --version` returns version info
- **Config:**
  ```bash
  git config --global user.name "Jim Calhoun"
  git config --global user.email "[jim's email]"
  git config --global core.autocrlf true
  ```

### 1.2 Node.js (LTS)
- **Install:** Node.js LTS (v20.x or v22.x) from https://nodejs.org/
- **Verify:**
  - `node --version`
  - `npm --version`
- **Global packages:**
  ```bash
  npm install -g pnpm
  npm install -g typescript
  npm install -g ts-node
  ```

### 1.3 Python
- **Install:** Python 3.11+ from https://www.python.org/downloads/
- **Important:** Check "Add Python to PATH" during install
- **Verify:** `python --version` and `pip --version`
- **Packages:**
  ```bash
  pip install uv
  pip install anthropic
  pip install notion-client
  ```

### 1.4 Bun (optional but recommended)
- **Install:** `powershell -c "irm bun.sh/install.ps1 | iex"`
- **Verify:** `bun --version`

## Phase 5: Environment Variables

### 5.1 Required API Keys
Set these as system environment variables:
- `ANTHROPIC_API_KEY` - For Claude API access
- `NOTION_API_KEY` - For Notion MCP
- `SUPABASE_URL` - If using Supabase
- `SUPABASE_ANON_KEY` - If using Supabase

### 5.2 How to Set (Windows)
```powershell
[System.Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-...", "User")
```
Or use System Properties > Environment Variables GUI.

## Troubleshooting

### PATH Issues
If any tool isn't recognized after install:
1. Open new terminal (existing terminals don't see PATH changes)
2. If still failing, verify in System Environment Variables GUI
3. Full path should include the `bin` or `cmd` subfolder
4. Restart Windows if changes don't propagate

### Permission Issues
Run terminal as Administrator for system-level installs.

### MCP Connection Issues
```bash
claude mcp list
claude mcp test [server-name]
```
